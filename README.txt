Para rodar o sistema, utilize o comando no diretório do arquivo:


npm install

Após isso, para rodar o arquivo na nuvem, utilizando a biblioteca Expo Go, utilize:

npm start

ou, em caso de Wifi do PC diferente do celular:

npm run tunnel
